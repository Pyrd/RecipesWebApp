# Template for Nuxt.js

## Features

- Dark Mode
- i18n
- API
- Toastify
- SEO Optimised
- SCSS Helpers
- Error management
- Vuetify

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## Deploy template to another git repositiory

```bash
$ npm run deploy
```
